# 4. 索引使用建议

## 4.1 索引列

1. 不为NULL
2. 频繁被查询
3. 常用于查询条件
4. 常用于连接查询条件
5. 频繁用于排序条件字段

## 4.6 索引查询失效

1. 使用select *
2. 创建了组合索引，但查询条件未遵守最左匹配原则;
3. 在索引列上进行计算、函数、类型转换等操作;
4. 以 % 开头的 LIKE 查询比如 like '%abc';
5. 查询条件中使用 or，且 or 的前后条件中有一个列没有索引，涉及的索引都不会被使用到;
6. 发生隐式转换;
