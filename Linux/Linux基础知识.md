# 1. 常见命令

目录切换：cd
目录内容显示：ll、ls
创建目录：mkdir
文件查找：find
显示当前目录路径：pwd
删除空文件夹：rmdir
删除文件及目录：rm、rm -r
文件复制：cp 
文件移动：mv
创建文件：touch
文件显示：
    cat：文件内容全部显示
    more：分屏显示或指定每次显示行数
    tail:动态显示

文件压缩：tar -zcvf xxx xxx
解压文件:tar -xvf xxx

机器运行状态：top
机器内存情况:free
机器磁盘使用情况：df -a
进程信息：ps 

ip地址：ifconfig
搜索命令：grep 要搜索的字符串 要搜索的文件 --color
杀死进程：kill -9 进程的pid
